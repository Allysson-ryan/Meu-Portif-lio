PetVida petshop

petshop idealizado e desenvolvido por mim com o objetivo de aplicar e consolidar meus aprendizados em HTML, CSS e JavaScript. a função dele é comprar produtos, agendar consultas, banho e tosa, além de hospedagem de animais. O site ainda está em desenvolvimento.

Hrml / Css / Javascript

Conversor de moedas

Este projeto é um conversor de moedas desenvolvido com HTML, CSS e JavaScript, usando a API Exchangerate para taxas de câmbio. É totalmente responsivo e inclui ícones de bandeiras para cada moeda, oferecendo uma interface intuitiva e atraente.

Hrml / Css / Javascript / API

Game JokenPô

É um jogo de Jokenpô (pedra, papel e tesoura) desenvolvido com HTML, CSS e JavaScript. O jogador escolhe entre pedra, papel ou tesoura e enfrenta a máquina, que faz uma escolha aleatória. O design é totalmente responsivo.

Hrml / Css / Javascript

Meu Portifólio

É um portfólio pessoal desenvolvido com HTML, CSS e JavaScript, utilizando a API Static Forms para gerenciar o envio de formulários de contato. É totalmente responsivo, com navegação intuitiva entre as seções que destacam habilidades e projetos realizados.
